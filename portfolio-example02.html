Travel Manager Application

Includes Design, UX, ERD and Frontend Development

Introduction

People sometimes book hotels or airlines over the Internet and then keep reservation information in files. In some cases, however, the file may be erased by a user mistake or the same file may continue to be downloaded by them. The app is an app that organizes files to reduce such mistakes. .

ERD & WireFrame

New Content

This application has a UI similar to OneNote. Users can view the first screen when they sign in to the application. When user press the Add button in the application, a new content creation pop-up appears.
